 - 线程

```python
import threading
import time


def sing():
    """唱歌5分钟"""
    for i in range(5):
        print("----正在唱01-----")
        time.sleep(1)

def dance():
    """跳舞5分钟"""
    for i in range(5):
        print("----正在跳舞01----")
        time.sleep(1)


def main():
    t1 = threading.Thread(target=sing)
    t2 = threading.Thread(target=dance)
    t1.start()
    t2.start()
    while True:
        length = len(threading.enumerate())
        print('当前运行的线程数为：%d' % length)
        sleep(1)
        if length == 1:
            break


if __name__ == '__main__':
    main()
```

 线程的执行顺序是不确定的， 抢占cpu， 所以可以有延迟操作，比如sleep
 
 主线程默认最后才会结束， 他会等待所有子线程完成。如果子线程没执行完，主线程死了，那么所有子线程全部死。
